# movieWebsite

## 使用框架和类库

本项目使用Node.js作为后台开发语言，MongoDB作为数据库。

后端使用的相关类库

- express4.x
- jade
- mongoose
- underscore
- path
- body-parser

前端使用的相关类库

- bootstrap
- jquery

## 项目介绍

本项目主要有4个页面，分别为：首页，列表页，新增录入页，详情页面。处理逻辑有新增，修改，删除操作。

## tips

clone或者直接下载本代码库后，需要安装Node.js和MongoDB，之后在项目根目录下运行 `npm install` && `bower install` 安装需要的类库。
